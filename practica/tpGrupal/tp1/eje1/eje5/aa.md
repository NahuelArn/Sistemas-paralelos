✔ ¿Qué diferencia se observa?

Se observa que usando float ambas raíces resultan iguales (2.00000), mientras que con double se obtienen valores distintos y correctos.

✔ ¿Por qué ocurre?

Esto ocurre porque el tipo float no tiene suficiente precisión para representar correctamente el valor del discriminante, ya que se produce una resta entre números muy cercanos.

Como consecuencia, el discriminante se aproxima a cero y se obtienen dos raíces iguales.

✔ Conclusión

El tipo double permite obtener resultados más precisos, mientras que float introduce errores numéricos en este tipo de cálculos.
